# 透過程式測試程式

## 套件與軟體
* Node.js 和 npm
* Mocha 和 Chai
* 瀏覽器

## 用 Mocha 寫測試 
* 初始化專案
  * npm init
* npm 安裝 Mocha 和 Chai
  * npm install mocha chai --save-dev
* 執行
  * ../test/index.html
